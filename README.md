# Impact of Dataset Quality and Data Leakage on Skin Cancer Detection

## 📌 Objective

This project investigates how dataset quality and data leakage affect the performance of deep learning models in skin cancer detection.

## 🧠 Key Idea

* Compare clean vs low-quality datasets
* Analyze impact of data leakage
* Evaluate misleading performance due to leakage

## 🛠️ Tools

* TensorFlow / Keras
* Python
* Google Colab

## 📊 Experiments

1. Clean dataset (no leakage)
2. Low-quality dataset
3. Dataset with leakage
4. Combined scenarios

## 📈 Expected Outcome

* Identify false high accuracy due to leakage
* Highlight importance of proper dataset handling
